# MOTU A16 Remote Volume Control Over TCP/IP
This code enables the use of a digital rotary encoder as a volume knob for ISC channels on the MOTU A16
The pin assignment is as follows:

GND : Pin 6 (GND)
+   : Pin 1 (3.3V)
SW  : NC
DT  : Pin 3 (GPIO 2)
CLK : Pin 5 (GPIO 3)
